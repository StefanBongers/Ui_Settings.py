Erste Schritte
==============
Nach dem erfolgreichen Login, startet **Geier** im Auswahlmodus. Um mit der Eingabe von Datensätzen zu beginnen, wähle
::

    Menü Eingabe > Daten eingeben
    ... oder Alt+e

Es öffnet sich das Haupteingabemenü. :doc:`Hier <eingabe>` findest Du eine detaillierte Beschreibung der
Eingabemaske.

.. note::

   Dieses Projekt wird fortwährend aktualisiert!
